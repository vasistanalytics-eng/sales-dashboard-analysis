# Sales Performance Dashboard for a Retail Superstore

## Objective
The objective of this project was to analyze retail sales data, identify trends in revenue and profit, and build an interactive Power BI dashboard for business decision-making.

## Dataset
This project uses a Superstore sales dataset containing order-level information such as sales, profit, category, sub-category, region, customer details, and dates.

## Tools Used
- Python
- Pandas
- Jupyter Notebook
- Power BI
- Matplotlib

## Project Workflow
1. Downloaded the raw dataset
2. Cleaned and prepared the data using Python
3. Converted date columns and standardized column names
4. Performed exploratory data analysis
5. Calculated key business metrics
6. Built an interactive Power BI dashboard
7. Summarized business insights

## Key Metrics
- Total Sales
- Total Profit
- Total Orders
- Average Order Value
- Monthly Sales Trend
- Category and Region Performance

## Key Insights
- The top-performing region contributed the highest share of sales.
- Some categories generated high sales but relatively lower profit.
- A small number of products drove a significant share of revenue.
- Sales showed monthly variation, indicating possible seasonal trends.
- Discounting may have impacted profitability in some cases.

## Files Included

## Conclusion
This project demonstrates skills in data cleaning, exploratory data analysis, data visualization, dashboard building, and business insight generation using Python and Power BI.
